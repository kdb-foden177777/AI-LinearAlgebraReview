# Marker PDF to Markdown - Environment Setup
# ============================================
#
# Prerequisites:
#   - Python 3.11 ~ 3.13 (Python 3.14 is NOT recommended, Pillow < 11 has no prebuilt wheel for 3.14)
#   - PyTorch (install manually first: https://pytorch.org/get-started/locally/)
#
# Setup steps:
#   1. python -m venv .venv
#   2. .venv\Scripts\activate   (Windows)  or  source .venv/bin/activate  (Mac/Linux)
#   3. pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
#   4. pip install -r requirements.txt
#   5. marker_single your_file.pdf --output_dir ./output --force_ocr
#
# Note on --force_ocr:
#   - Recommended for math-heavy PDFs (converts inline math to LaTeX)
#   - Skip this flag for plain-text PDFs (much faster, uses embedded text directly)
#
# Total download size: ~2-3 GB (mostly model weights, downloaded on first run)
